Interesting Thingys #2

Jan 23rd, 2012 | Comments

Interesting Thingys is a gazette in the blog that summarizes a week of blog articles, links, specifications, demos, news or more general “thingys” that we stumble upon in our journey through the world wide web.

Server/WebStack

  • WebSockets, SPDY Protocol, HTTP2.0 - .. “Tommorrows technology for todays web!”.

Java

  • RESTful Webservices with Spring/Java - Extensive tutorial in 7(!) parts.

HTML5

  • HTML5 Reference Poster
  • Presentation about “The Bleeding Edge of HTML5” - . New CSS techniques, Javascript APIs, Multimedia thingys .. Works mostly only in Chrome and Safari. Very intersting!.

Mobile

  • iOS UI Patterns, - Overview of UI Patterns in iOS Apps.
  • Android UI Patterns
  • VIM for the iPad. - Useful: Nope. Geeky: Yep!
  • Android Terminal IDE. - Useful: Yep. Geeky: Yep!

JavaScript/CSS

  • The “Hello World” of Web-Programming - How to build a Blog App using NodeJS, MongoDB & CoffeeScript (using a BDD approach).
  • thorax - Application Framework for Backbone Apps. (including underscore, zepto, handlebars, stylus, …).
  • jQuery Collapse - Expanding & Collapsing Content (with CookieSupport, ARIA Compliant, lightweight, …).
  • nodejs & aws - nodejs Library to talk to Amazon Webservices.
  • WAT?! - 10min lightning talk about “WATs”* of Ruby & Javascript, very entertaining

Ruby

  • vesperapps - New web app framework based on Sinatra.
  • subtle - A Window Tiling Manager for Linux, with a Ruby-Scripting Layer.
